## Getting Started

Simply clone the repository, and run Art_Origin.py with python 3.5+


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/cdwr/Art-Generator.git
   ```
2. Install Pillow
    ```sh
    pip install PIL
    ```
3. Run
    ```sh
    python3 Art_Origin.py
    ```
